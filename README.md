# Personal CV Website (HTML & CSS)

A simple one-page personal CV website created using HTML and CSS. This project demonstrates front-end skills in layout, design, and styling.

## Purpose
- Practice structuring a web page
- Learn basic CSS styling techniques
- Create a clean, readable one-page CV

## Features
- Sectioned layout (Education, Skills, Projects, Contact)
- Simple, responsive design (basic HTML & CSS)
- Visual styling with CSS

## Folder Structure
- `index.html`          → Main CV page
- `style.css`           → CSS styling file
- `screenshots/`        → Preview images of the CV page

## Technologies Used
- HTML
- CSS
